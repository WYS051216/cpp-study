# C++ 现代特性笔记：移动语义与 For-each 循环

## 一、 移动语义 (Move Semantics)

### 1. 核心概念
移动语义允许将资源（如堆内存、系统对象、文件句柄等）**直接从一个对象转移（偷取）到另一个对象**。这样可以极大地减少不必要的临时对象的创建、深拷贝以及销毁，从而榨干硬件的极致性能。

在 C++ 中，自定义类如果要实现移动语义，通常需要定义：
1. **移动构造函数 (Move Constructor)**
2. **移动赋值操作符 (Move Assignment Operator)**

### 2. 实现移动语义的三大硬性注意点
* **① 参数（右值）的符号必须是 `&&`**：必须是右值引用，这样编译器才能精准匹配到临时对象或被显式标记的资源。
* **② 参数绝对不能是常量 (`const`)**：因为移动的本质是“偷取”资源，我们需要在函数内部修改这个右值（比如将其指针清空）。如果加了 `const`，资源就无法被转移。
* **③ 必须修改原右值的资源链接和标记**：在夺取资源的控制权后，**必须将原右值的指针置为 `nullptr`**（断开链接）。否则，当该右值生命周期结束触发析构函数时，就会把原本已经转移给新对象的堆内存给释放掉，导致新对象直接崩溃。

### 3. `std::move` 的本质
标准库函数 `std::move` **并不能移动任何东西**，它的本质是一个强制类型转换。它唯一的作用是**将一个左值强制转换为右值引用**，从而像拿到了“通行证”一样，允许编译器去触发移动语义。

### 4. 代码示例
```cpp
#include <iostream>
#include <utility> // std::move 所需头文件

class ResourceManager {
public:
    int* data;

    // 构造函数
    ResourceManager() : data(new int[100]) { std::cout << "申请 100 个 int 内存\n"; }
    ~ResourceManager() {
        if (data != nullptr) {
            delete[] data;
            std::cout << "释放内存\n";
        }
    }

    // 移动构造函数 (满足上述三大注意点)
    ResourceManager(ResourceManager&& other) noexcept : data(other.data) { // ① 符号为 &&
        other.data = nullptr; // ② & ③ 修改原右值的标记，防止旧对象析构时释放资源
        std::cout << "触发移动构造，完美转移资源！\n";
    }
};

int main() {
    ResourceManager a; 
    // 将左值 a 强转为右值，交给移动构造函数
    ResourceManager b = std::move(a); 
    return 0;
}
```

## 二、 For-each 循环（范围 For 循环 / Range-based for loop）

### 1. 概念补充
For-each 循环是 C++11 引入的一种更安全、更简洁的遍历容器（如 `std::vector`, `std::map`, 甚至原生数组）的语法推导式。

* **底层原理**：它是一个典型的“语法糖”。编译器在编译时，会自动将其展开为利用容器的 `begin()` 和 `end()` 迭代器（Iterator）进行遍历的传统 `for` 循环。因此，**任何实现了 `begin()` 和 `end()` 方法的自定义类，都可以直接使用 For-each 进行遍历**。

### 2. 核心语法与推导式分类
在实际开发中，我们通常结合 `auto` 关键字来让编译器自动推导容器内元素的类型。根据不同的需求，主要分为以下三种写法：

| 写法格式 | 推导效果 | 适用场景 |
| :--- | :--- | :--- |
| `for (auto x : vec)` | **值传递（拷贝）** | 只读，且元素体积很小（如 `int`, `char`） |
| `for (const auto& x : vec)` | **常量引用（只读且0拷贝）** | **推荐首选**。只读，且元素体积较大（如 `std::string`, 自定义类） |
| `for (auto& x : vec)` | **非常量引用（可修改）** | **需要修改容器内元素的值**时使用 |

### 3. 代码示例
```cpp
#include <iostream>
#include <vector>
#include <string>

int main() {
    std::vector<std::string> words = {"Hello", "Modern", "C++"};

    // 写法 1：常量引用遍历（高效只读，不发生 1GB 数据的复制）
    std::cout << "--- 常量引用遍历 ---" << std::endl;
    for (const auto& word : words) {
        std::cout << word << " "; // 只能读取，不能修改 word
    }
    std::cout << "\n\n";

    // 写法 2：非常量引用遍历（可以改变容器里的内容）
    std::cout << "--- 修改容器元素 ---" << std::endl;
    for (auto& word : words) {
        word += "!"; // 在每个单词后面加个感叹号
    }

    // 写法 3：值传递遍历（此时 word 是复制出来的一个副本）
    for (auto word : words) {
        std::cout << word << " "; // 输出: Hello! Modern! C++!
    }
    std::cout << "\n";

    return 0;
}
```
