# C++ 核心学习笔记：内联函数与函数重载

## 一、 内联函数 (Inline Function)

### 1. 产生的背景与痛点
*   **栈空间消耗：** 在 C/C++ 中，栈空间（存放程序局部数据、函数内数据的内存）是有限的。如果某些函数被频繁调用，会导致不断有函数入栈，消耗大量栈内存。如果发生死循环或极深的递归调用，最终会导致栈内存枯竭（Stack Overflow，栈溢出），造成程序崩溃。
*   **调用开销：** 函数是一个可重复使用的代码块，但调用函数是有**时间和空间开销**的。
    *   **准备阶段（压栈）：** 需要将实参、局部变量、返回地址以及若干寄存器压入栈中。
    *   **执行阶段：** 跳转去执行函数体。
    *   **清理阶段（出栈）：** 执行完毕后清理现场，将压入栈的数据出栈，返回调用位置继续执行。

### 2. 内联函数的原理
为了消除上述频繁调用函数带来的时空开销，C++ 引入了 `inline` 修饰符。
*   **工作机制：** 在**编译时**，编译器会将函数调用的地方直接用函数体的代码进行替换（展开），类似于 C 语言中的宏展开（Macro）。这种直接在调用处嵌入函数体的函数，就称为内联函数（内置函数）。

### 3. 定义与示例
*   **定义规则：** 关键字 `inline` **必须**与函数定义体放在一起才能生效。仅仅将 `inline` 放在函数声明前面是不起任何作用的。

**示例代码：**
```cpp
#include <iostream>
using namespace std;

// inline 必须和函数体绑定在一起
inline int add(int a, int b) {
    return a + b;
}

int main() {
    // 编译时，此处相当于直接被替换为: int result = 10 + 20;
    int result = add(10, 20); 
    cout << "Result: " << result << endl;
    return 0;
}
```

### 4. 使用限制
内联函数本质上是“空间换时间”，因此有严格的使用限制：
1.  **代码必须简单：** 只适合函数体内代码非常短小精悍的函数。
2.  **无复杂控制流：** 不能包含复杂的结构控制语句（例如 `while`、`switch`、`for` 循环）。
3.  **禁止递归：** 内联函数本身不能是直接递归函数（内部不能自己调用自己），否则会导致无限展开。

### 5. 内联函数 vs 普通函数对比总结

| 特性对比 | 普通函数 | 内联函数 (Inline) |
| :--- | :--- | :--- |
| **执行过程** | 跳跃寻址 $\rightarrow$ 执行 $\rightarrow$ 返回 | 无寻址过程，直接原地展开代码 |
| **内存拷贝** | 代码段中**始终只有 1 个**拷贝 | 调用 N 次，代码段就会有 **N 个**拷贝 |
| **栈内存开销** | 大（需要频繁压栈、出栈） | 极小（借用调用者的栈空间） |

## 二、 函数重载 (Function Overloading)

### 1. 核心概念
C++ 允许我们在同一个作用域内，为多个功能相似的函数**设置相同的函数名字**，前提是这些函数的**参数列表必须不一致**。这极大地提高了代码的可读性和复用性。

### 2. 重载的四大规则
1.  **同名：** 函数名称必须完全相同。
2.  **参数不同（核心）：** 参数列表必须不同（可以是**个数不同**、**类型不同**、或者**参数排列顺序不同**）。
3.  **返回类型随意：** 函数的返回类型可以相同，也可以不同。
4.  **返回类型不能作为判定依据：** **仅仅**返回类型不同，不足以构成函数的重载，编译器会报错（产生二义性）。

### 3. 示例代码
以下代码展示了三种合法的函数重载方式：

```cpp
#include <iostream>
using namespace std;

// 1. 基础函数
void printInfo(int age) {
    cout << "Age is: " << age << endl;
}

// 2. 重载：参数【类型】不同
void printInfo(double weight) {
    cout << "Weight is: " << weight << endl;
}

// 3. 重载：参数【个数】不同
void printInfo(int age, double weight) {
    cout << "Age: " << age << ", Weight: " << weight << endl;
}

// 4. 重载：参数【顺序】不同
void printInfo(double weight, int age) {
    cout << "Weight: " << weight << ", Age: " << age << endl;
}

/* 
// 错误示范：仅仅返回值不同，不能构成重载！
int printInfo(int age) {
    return age; 
}
*/

int main() {
    printInfo(25);          // 调用基础函数 (int)
    printInfo(65.5);        // 调用重载 2 (double)
    printInfo(25, 65.5);    // 调用重载 3 (int, double)
    printInfo(65.5, 25);    // 调用重载 4 (double, int)
    return 0;
}
```
