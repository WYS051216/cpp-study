# C/C++ 核心学习笔记：共用体（联合体）与枚举类型

## 一、 共用体 (Union / 联合体)

### 1. 核心概念
*   **定义：** 共用体可以存放任意数据类型的组合/集合。在 C/C++ 中，共用体也叫**联合体**。
*   **内存共享特性（最重要）：** 结构体的各个成员会占用不同的内存，互相之间没有影响；而**共用体的所有成员占用同一段内存**。修改其中一个成员，会**覆盖**并影响其余所有成员。
*   **内存大小：** 共用体占用的内存等于其中**最长的类型/成员**所占用的内存。



### 2. 语法与访问
```cpp
// 声明语法
union UnionName {
    type1 field1;
    type2 field2;
    type3 field3;
};

// 访问语法
// UnionNameVar.field1;
```

### 3. 共用体与结构体的深度对比

| 特性对比 | 结构体 (Struct) | 共用体 (Union) |
| :--- | :--- | :--- |
| **内存占用** | 成员拥有各自独立的内存。总内存 $\ge$ 所有成员内存之和（存在内存对齐）。 | 所有成员共享同一段内存。总内存 = 最大成员的内存。 |
| **相互影响** | 修改一个成员，**不影响**其他成员。 | 修改一个成员，**会覆盖**并影响其他成员的值。 |
| **数据保存** | 同一时刻可以保存**所有**成员的值。 | 同一时刻只能保存**一个**成员的值（使用了内存覆盖技术）。 |

### 4. 共用体代码示例

```cpp
#include <iostream>
using namespace std;

// 定义共用体：最大成员是 double (通常占 8 字节)
union MyData {
    int i;
    char c;
    double d;
};

int main() {
    MyData data;

    // 1. 赋值整型
    data.i = 65; 
    cout << "赋值 data.i = 65 后:" << endl;
    cout << "data.i = " << data.i << endl;
    cout << "data.c = " << data.c << "  // 65 对应的 ASCII 码是 'A'，因为共享内存" << endl;

    cout << "-------------------" << endl;

    // 2. 赋值浮点型 (此时会覆盖之前的 65)
    data.d = 3.14159;
    cout << "赋值 data.d = 3.14159 后:" << endl;
    cout << "data.d = " << data.d << endl;
    cout << "data.i = " << data.i << " // 原来的 65 被覆盖，现在是一个无意义的错乱数字" << endl;

    return 0;
}
```

## 二、 枚举类型 (Enum)

### 1. 核心概念
枚举常量或枚举成员是以**标识符形式**表示的整型量，用于表示变量的有限个取值集合。
*   **语法：** `enum <类型名> { <枚举常量表> };`
*   枚举常量表列出了枚举类型的所有取值，以 `,` 间隔，且名称必须各不相同。
*   **限制：** 枚举常量只能以“标识符”（英文单词/字母）形式表示，绝对不能是 `1`、`'a'` 等文字常量。

### 2. 【重点补充】枚举常量指定整数值的原则
默认情况下，枚举常量会隐式地与整数值相对应：
1.  **默认起始值为 0：** 如果没有显式赋值，第一个枚举常量的值默认为 `0`。
2.  **依次递增 1：** 后续的枚举常量值在前面的基础上自动加 `1`。
3.  **支持显式赋值：** 可以给任意枚举常量强制指定一个整数值，其后面的常量会在这个指定值的基础上依次加 `1`。

### 3. 枚举变量的使用与特性
*   **声明变量：** `EnumName enumVarName;`
*   **取值限制：** 枚举变量的值只能取枚举常量表中所列的值（本质上是整型数的一个子集）。
*   **内存大小：** 占用内存的大小与普通整型数（`int`）相同（通常为 4 字节）。
*   **运算支持：** 枚举变量只能参与**赋值**、**关系运算**（比较大小）以及**输出操作**。参与运算时，使用的是其底层的整数值。

### 4. 枚举类型代码示例

```cpp
#include <iostream>
using namespace std;

// 定义枚举类型并演示赋值原则
enum Weekday {
    MON,          // 默认值为 0
    TUE,          // 自动递增，值为 1
    WED = 10,     // 显式赋值为 10
    THU,          // 在 10 的基础上递增，值为 11
    FRI,          // 值为 12
    SAT,
    SUN
};

int main() {
    // 声明并使用枚举变量
    Weekday today = TUE;
    Weekday payday = FRI;

    // 输出操作（输出的是底层的整数值）
    cout << "today (TUE) 的值为: " << today << endl;
    cout << "payday (FRI) 的值为: " << payday << endl;

    // 关系运算操作
    if (payday > today) {
        cout << "发薪日还在后面呢！" << endl;
    }

    // 赋值操作：只能赋枚举表里的值
    today = WED; 
    cout << "修改后 today (WED) 的值为: " << today << endl;

    return 0;
}
```
