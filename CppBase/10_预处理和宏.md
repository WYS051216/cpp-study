# C++ 核心学习笔记：预处理指令 (Preprocessor)

## 一、 预处理的核心概念

1. **什么是预处理？**
   预处理是由**预处理程序**负责完成的。在对源文件进行正式编译（第一遍词法扫描和语法分析）**之前**，系统会自动调用预处理程序，对代码中所有的预处理指令进行处理。处理完毕后，才会自动进入真正的编译阶段。
2. **为什么需要预处理？**
   C++ 提供了多种预处理功能（如宏定义、文件包含、条件编译等）。合理使用它们，可以使程序：
   * 便于阅读、修改、移植和测试。
   * 极大地利于**模块化程序设计**。
3. **基本语法规则：**
   * 预处理命令主要是处理以 **`#`** 开头的指令（例如 `#include <iostream>`）。
   * 预处理命令一般要放在**所有函数之外**，并且通常习惯放在源文件的**最前面**（头部）。

## 二、 文件包含类 (File Inclusion)

用于将另一个源代码文件或头文件的内容，原封不动地“复制粘贴”到当前位置。

* **`#include <...>`**：引用编译器类库路径里的标准头文件。
* **`#include "..."`**：优先引用当前项目相对路径中的自定义头文件。

**代码示例：**
```cpp
#include <iostream>      // 包含标准输入输出流库
#include "my_math.h"     // 包含自定义的头文件

int main() {
    std::cout << "文件包含成功！" << std::endl;
    return 0;
}
```

## 三、 条件编译类 (Conditional Compilation)

条件编译允许编译器根据特定的条件，决定是否编译某一段代码。这在跨平台开发、调试代码、版本控制时非常有用。

* **`#if`**：如果给定条件为真，则编译下方代码。
* **`#ifdef`**：如果某个宏**已经定义**，则编译下方代码。
* **`#ifndef`**：如果某个宏**没有定义**，则编译下方代码（常用来防止头文件重复包含）。
* **`#elif`**：相当于 `else if`，如果前面的 `#if` 为假，且当前条件为真，则编译。
* **`#else`**：如果前面的所有条件都为假，则编译此处代码。
* **`#endif`**：结束条件编译块。

**代码示例：**
```cpp
#include <iostream>
using namespace std;

#define DEBUG_MODE 1  // 开启调试模式
#define VERSION 2     // 设定版本号

int main() {
    // 1. #ifdef 示例
#ifdef DEBUG_MODE
    cout << "[DEBUG] 调试模式已开启，正在输出日志..." << endl;
#endif

    // 2. #ifndef 示例
#ifndef RELEASE_MODE
    cout << "当前不是正式发布版本。" << endl;
#endif

    // 3. #if / #elif / #endif 示例
#if VERSION == 1
    cout << "正在运行 V1.0 基础版逻辑。" << endl;
#elif VERSION == 2
    cout << "正在运行 V2.0 升级版逻辑。" << endl;
#else
    cout << "未知版本。" << endl;
#endif

    return 0;
}
```

## 四、 宏定义类 (Macro Definition)

宏的本质是**纯文本替换**（也叫宏展开）。在编译预处理时，预处理程序会自动将程序中出现的所有“宏名”，无脑替换成宏定义中的“字符串/表达式”。

### 1. 无参数宏定义
通常用于定义常量。
* **语法**：`#define 标识符(宏名) 表达式或常量值`

**代码示例：**
```cpp
#include <iostream>
#define PI 3.14159       // 定义无参宏
#define MAX_BUFFER 1024

int main() {
    double radius = 2.0;
    // 预处理阶段，PI 会被自动替换为 3.14159
    double area = PI * radius * radius; 
    std::cout << "面积: " << area << std::endl;
    return 0;
}
```

### 2. 带参数宏定义 (宏函数)
C++ 中的宏函数跟普通函数类似，但**宏函数的参数是没有数据类型的**，因为它仅仅做字符串替换。

* **⚠️ 避坑指南：** 编写宏函数时，**务必给所有参数以及整个表达式加上括号**，以防止运算符优先级带来的逻辑错误。

**代码示例：**
```cpp
#include <iostream>

// 定义宏函数：求两个数的最大值
// 严谨写法：参数和外层都加了括号
#define MAX(a, b) ((a) > (b) ? (a) : (b))

// 定义宏函数：自增操作 (此处原笔记误放到了无参宏中)
#define INC(x) ((x)++) 

int main() {
    int x = 10;
    int y = 20;

    // 预处理时被替换为：int biggest = ((10) > (20) ? (10) : (20));
    int biggest = MAX(x, y); 
    std::cout << "最大值是: " << biggest << std::endl;

    // 预处理时被替换为：((x)++);
    INC(x); 
    std::cout << "x 自增后的值: " << x << std::endl; // 输出 11

    return 0;
}
```

### 3. 取消宏定义 (`#undef`)
如果在后续代码中不再需要某个宏，或者想重新定义它，可以使用 `#undef` 提前结束它的作用域。
* **语法**：`#undef 宏名`

**代码示例：**
```cpp
#include <iostream>

#define MODE 1

int main() {
    std::cout << "Mode is: " << MODE << std::endl;

#undef MODE // 从这一行开始，MODE 这个宏失效了

// 下面这行代码如果取消注释，编译器会报错，因为 MODE 已经被取消定义了
// std::cout << "Mode is: " << MODE << std::endl; 

    return 0;
}
```
