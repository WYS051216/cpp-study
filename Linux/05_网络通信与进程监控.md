## 1. 网络基础配置
### 1.1 IP 与主机名
- **查看 IP**：`ip addr` (或 `ip a`)、`ifconfig`。`127.0.0.1` 指代本机。
- **查看/修改主机名**：`hostname` / `hostnamectl set-hostname 名字`

### 1.2 域名解析与映射
- **域名解析**：将易读的域名（如 baidu.com）转换为 IP 地址的过程（DNS）。
- **本地映射 (`/etc/hosts`)**：在文件末尾添加 `IP地址 域名/主机名`，可在本地强制解析，常用于测试环境搭建。

## 2. 网络测试与请求
- **`ping` (测连通性)**：`ping [-c 测试次数] IP或域名`。
- **`wget` (下载文件)**：`wget [-b] URL`（`-b` 后台下载）。
- **`curl` (发送 HTTP 请求)**：`curl [-O / -o 自定义文件名] URL`。

## 3. 端口与进程管理
### 3.1 查看端口占用
端口是设备与外界通讯的出入口（0~65535）。运行服务端程序必须绑定特定端口。
- **查看占用**：`netstat -anp | grep 端口号`

### 3.2 进程管理
每个运行的程序都会分配一个独有的进程 ID (PID)。
- **查看进程 (`ps`)**：`ps -ef | grep 关键字`（`-e` 全部进程，`-f` 格式化展示）。
- **关闭进程 (`kill`)**：`kill [-9] 进程ID`（`-9` 表示系统强制杀死）。

## 4. 主机状态与性能监控
### 4.1 综合资源监控 (`top`)
实时查看 CPU、内存及进程排行（默认 5 秒刷新）。
- **交互按键**：`P` (按 CPU 排序), `M` (按内存排序), `1` (展开所有 CPU 核心), `q` (退出)。

### 4.2 磁盘与 I/O 监控
- **磁盘使用 (`df`)**：`df -h`（易读单位显示硬盘空间）。
- **I/O 监控 (`iostat`)**：`iostat [-x] [间隔时间] [刷新次数]`（查看 CPU 与磁盘 I/O 详情）。

### 4.3 网络状态监控 (`sar`)
- **语法**：`sar -n DEV 刷新间隔 查看次数`（查看网络接口网卡的流量和收发包统计）。
