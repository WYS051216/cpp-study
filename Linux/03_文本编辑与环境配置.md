## 1. vi/vim 编辑器
Linux 中最经典的文本编辑器，具有代码高亮功能。语法：`vi/vim 文件路径`。

### 1.1 三种工作模式
1. **命令模式 (Normal Mode)**：默认模式，进行复制、粘贴、删除。
2. **输入模式 (Insert Mode)**：对文件内容进行具体编辑。
3. **底线命令模式 (Command-line Mode)**：用于保存、退出、搜索等。

### 1.2 常用快捷键
- **命令模式下**：
  - `i` / `a` / `o`：在光标前 / 光标后 / 换行 进入输入模式。
  - `dd` (删除/剪切当前行), `yy` (复制当前行), `p` (粘贴), `u` (撤销)。
  - `gg` (跳到开头), `G` (跳到末尾)。
  - `/关键字`：搜索关键字（`n` 下一个，`N` 上一个）。
- **底线命令模式**（按 `:` 进入）：
  - `:w` (保存), `:q` (退出), `:wq` (保存并退出), `:q!` (强制退出不保存)。

## 2. 环境变量与 PATH
环境变量是操作系统运行时的关键配置信息。
- **查看**：`env` 查看所有；`echo $变量名` 取单个值（如 `echo $PATH`）。

### 2.1 自行设置环境变量
- **临时设置**：`export 变量名=值`（终端关闭失效）。
- **永久生效**：
  - 当前用户：写入 `~/.bashrc`
  - 所有用户：写入 `/etc/profile`
  - **立刻生效**：执行 `source ~/.bashrc` 或重新登录。

### 2.2 自定义 PATH 变量
`PATH` 记录了可执行程序的寻找目录。让自己的程序随处可运行：
- 在 `~/.bashrc` 末尾添加：`export PATH=$PATH:/你的/自定义/目录路径`

## 3. 日期和时区
### 3.1 `date` 命令
- **语法**：`date [-d] [+格式化字符]`
- **格式化字符**：`%Y` (年), `%m` (月), `%d` (日), `%H` (时), `%M` (分), `%S` (秒), `%s` (时间戳)。
- **示例**：`date "+%Y-%m-%d %H:%M:%S"`（输出 YYYY-MM-DD HH:MM:SS）。

### 3.2 `ntp` 程序
- 自动校准系统时间。配置好后在后台运行，通过网络时间服务器自动同步本机时间。
