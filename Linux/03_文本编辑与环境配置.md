# Linux vi/vim 编辑器笔记

## 1. vi/vim 基本用法

`vi/vim` 是 Linux 中常用的文本编辑器。

基本语法：

```bash
vim 文件名
```

示例：

```bash
vim test.c
```

如果文件存在，则打开文件；如果文件不存在，则新建文件。

---

## 1.2. vi/vim 三种常用模式

| 模式 | 作用 |
|---|---|
| 命令模式 | 移动光标、复制、删除、粘贴、撤销 |
| 输入模式 | 编辑文件内容 |
| 底线命令模式 | 保存、退出、搜索、替换 |

进入 vim 后，默认处于**命令模式**。

---

## 1.3. 模式切换

| 操作 | 作用 |
|---|---|
| `i` | 在光标前进入输入模式 |
| `a` | 在光标后进入输入模式 |
| `o` | 在当前行的下一行新建一行并进入输入模式 |
| `Esc` | 从输入模式回到命令模式 |
| `:` | 从命令模式进入底线命令模式 |

常用流程：

```text
vim 文件名
按 i 开始编辑
按 Esc 退出编辑
输入 :wq 保存并退出
```

---

## 1.4. 保存和退出

先按 `Esc` 回到命令模式，再输入以下命令。

| 命令 | 作用 |
|---|---|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 强制退出，不保存 |
| `:wq!` | 强制保存并退出 |

最常用：

```vim
:wq
```

不想保存直接退出：

```vim
:q!
```

## 1.5. 光标移动

在命令模式下使用。

| 命令 | 作用 |
|---|---|
| `h` | 左移 |
| `j` | 下移 |
| `k` | 上移 |
| `l` | 右移 |
| `gg` | 跳到文件开头 |
| `G` | 跳到文件末尾 |
| `数字G` | 跳到指定行 |
| `0` | 跳到当前行行首 |
| `$` | 跳到当前行行尾 |

示例：

```text
10G
```

表示跳到第 10 行。

## 1.6. 删除、复制、粘贴

在命令模式下使用。

| 命令 | 作用 |
|---|---|
| `x` | 删除光标所在字符 |
| `dd` | 删除当前行 |
| `数字dd` | 删除多行 |
| `yy` | 复制当前行 |
| `数字yy` | 复制多行 |
| `p` | 粘贴到当前行下面 |
| `P` | 粘贴到当前行上面 |

示例：

```text
dd
```

删除当前行。

```text
3dd
```

删除当前行开始的 3 行。

```text
yy
p
```

复制当前行并粘贴到下一行。

## 1.7. 撤销和恢复

在命令模式下使用。

| 命令 | 作用 |
|---|---|
| `u` | 撤销上一步操作 |
| `Ctrl + r` | 恢复撤销的操作 |

## 1.8. 搜索内容

在命令模式下使用。

| 命令 | 作用 |
|---|---|
| `/关键字` | 向下搜索 |
| `?关键字` | 向上搜索 |
| `n` | 查找下一个 |
| `N` | 查找上一个 |

示例：

```vim
/main
```

表示搜索 `main`。

取消搜索高亮：

```vim
:noh
```

---

## 1.9. 替换内容

在底线命令模式下使用。

### 替换当前行第一个匹配项

```vim
:s/旧内容/新内容/
```

### 替换当前行所有匹配项

```vim
:s/旧内容/新内容/g
```

### 替换全文所有匹配项

```vim
:%s/旧内容/新内容/g
```

示例：

```vim
:%s/hello/world/g
```

表示把全文所有 `hello` 替换为 `world`。

## 2. 环境变量与 PATH
环境变量是操作系统运行时的关键配置信息。
- **查看**：`env` 查看所有；`echo $变量名` 取单个值（如 `echo $PATH`）。

### 2.1 自行设置环境变量
- **临时设置**：`export 变量名=值`（终端关闭失效）。
- **永久生效**：
  - 当前用户：写入 `~/.bashrc`
  - 所有用户：写入 `/etc/profile`
  - **立刻生效**：执行 `source ~/.bashrc` 或重新登录。

### 2.2 自定义 PATH 变量
`PATH` 记录了可执行程序的寻找目录。让自己的程序随处可运行：
- 在 `~/.bashrc` 末尾添加：`export PATH=$PATH:/你的/自定义/目录路径`

## 3. 日期和时区
### 3.1 `date` 命令
- **语法**：`date [-d] [+格式化字符]`
- **格式化字符**：`%Y` (年), `%m` (月), `%d` (日), `%H` (时), `%M` (分), `%S` (秒), `%s` (时间戳)。
- **示例**：`date "+%Y-%m-%d %H:%M:%S"`（输出 YYYY-MM-DD HH:MM:SS）。

### 3.2 `ntp` 程序
- 自动校准系统时间。配置好后在后台运行，通过网络时间服务器自动同步本机时间。
