# 📝 Linux 基础实操与核心命令入门

## 1. Linux 目录结构与路径观念

与 Windows 不同，Linux **没有盘符**（C盘、D盘）的概念。
整个系统只有一个**根目录 `/`**，所有的文件和目录都在它之下呈树状展开。在 Linux 系统中，路径之间的层级关系用 `/` 来表示。

### 1.1 绝对路径 vs 相对路径
* **绝对路径**：以根目录 `/` 为起点描述路径，**必定以 `/` 开头**。（例如：`/home/itheima/Desktop`）
* **相对路径**：以当前所在目录为起点描述路径，**无需以 `/` 开头**。（例如：`Desktop/test`）

### 1.2 特殊路径符号 (极高频)
* `.` ：表示当前目录。
* `..` ：表示上一级目录（`cd ..` 回退一级，`cd ../..` 回退两级）。
* `~` ：表示当前用户的 HOME 目录（`cd ~/Desktop` 可直接切到桌面的绝对领域）。

## 2. Linux 命令基础语法
命令行（终端）以纯“字符”的形式操作系统。一个命令就是一个 Linux 程序。
**标准格式**：`command [-options] [parameter]`
* **command**：命令本身（如 `ls`、`cd`）。
* **-options**：可选选项，控制命令的行为细节（如 `-l`、`-a`）。
* **parameter**：命令的参数，指向操作的目标路径或文件。

*(例：`ls -l /home/itheima` 中，`ls` 是命令，`-l` 是选项，`/home/itheima` 是参数。)*

## 3. 目录与导航命令

### 3.1 `pwd` (Print Work Directory)
* **作用**：查看当前所在的工作目录的绝对路径。
* **语法**：`pwd` （无选项，无参数，迷路时必备）。

### 3.2 `cd` (Change Directory)
* **作用**：切换当前工作目录。
* **语法**：`cd [Linux路径]`
* *注：如果不加任何参数直接执行 `cd`，会直接瞬间回城到用户的 HOME 目录。*

### 3.3 `ls` (List)
* **作用**：列出目录下的内容。默认平铺展示当前目录。
* **核心选项**：
    * `-a` (all)：列出全部文件，包括以 `.` 开头的**隐藏文件/文件夹**。
    * `-l` (list)：以列表（竖向）形式展示，并显示权限、大小、修改时间等详细信息。
    * `-h` (human-readable)：以易读形式显示大小（如 K, M, G），**必须搭配 `-l` 使用**（如 `ls -lh`）。
    * *注：选项可以随意组合，如 `ls -la` 或 `ls -al` 效果相同。*

## 4. 文件与文件夹操作命令

### 4.1 创建操作
* **创建文件夹 `mkdir` (Make Directory)**
    * **语法**：`mkdir [-p] Linux路径`
    * **选项**：`-p` 用于自动创建不存在的父目录（例如：`mkdir -p a/b/c` 可以一次性把三层目录全建好）。
* **创建空文件 `touch`**
    * **语法**：`touch Linux路径` （如 `touch test.txt`）。

### 4.2 查看文件内容
* **全部查看 `cat`**：一次性将文件所有内容输出到屏幕。语法：`cat Linux路径`。
* **翻页查看 `more`**：适合查看超大文件。可以按空格键一页页翻，按 `q` 退出。语法：`more Linux路径`。

### 4.3 复制与移动
* **复制 `cp` (Copy)**
    * **语法**：`cp [-r] 参数1(被复制的源) 参数2(目标位置)`
    * **选项**：复制文件夹时**必须**加 `-r`（递归）选项。
* **移动/重命名 `mv` (Move)**
    * **语法**：`mv 参数1(被移动的源) 参数2(目标位置)`
    * *注：如果目标路径是一个不存在的名字，`mv` 会顺手完成**重命名**操作。*

### 4.4 删除操作 (高危 ⚠️)
* **删除 `rm` (Remove)**
    * **语法**：`rm [-r -f] 参数1 参数2 ... 参数n`
    * **选项**：
        * `-r`：递归删除（用于删除文件夹）。
        * `-f` (force)：强制删除，绝不弹窗确认。
* **通配符 `*` (模糊匹配神器)**
    * `*` 表示匹配任意数量的任意字符。
    * `test*`：匹配任何以 test 开头的内容。
    * `*test`：匹配任何以 test 结尾的内容（如 `rm *.txt` 删光所有文本文件）。
    * `*test*`：匹配任何包含 test 的内容。
