# Linux 线程基础笔记

## 1. 线程概述
- 线程是**轻量级进程（LWP）**，在 Linux 环境下，线程本质上通过轻量级进程实现。
- 进程是**资源分配的最小单位**，线程是**操作系统调度执行的最小单位**。

## 2. 线程与进程的区别

### 2.1 地址空间与资源
- **进程**：拥有独立的虚拟地址空间，资源分配独立。
- **线程**：多个线程共享同一进程的地址空间（代码段、堆区、全局数据区、打开的文件等），但每个线程有独立的栈区和寄存器。

线程的优势：
- 更节省系统资源
- 创建和退出速度更快
- 上下文切换开销更小

### 2.2 执行与调度
- **进程**：是资源分配单位，一个进程对应一个虚拟地址空间。
- **线程**：是程序执行单位，一个进程中可以有多个线程，每个线程都可被调度执行。

## 3. 线程创建

### 3.1 线程 ID
- 类型：`pthread_t`
- 用于唯一标识线程
- 获取当前线程 ID：

```c
pthread_t pthread_self(void);
```

### 3.2 创建线程函数
```c
int pthread_create(
    pthread_t *thread,
    const pthread_attr_t *attr,
    void *(*start_routine)(void *),
    void *arg
);
```

参数说明：
- `thread`：传出参数，用于保存新线程 ID
- `attr`：线程属性，一般写 `NULL`
- `start_routine`：线程执行函数
- `arg`：传递给线程函数的参数

注意：
- 线程函数必须写成 `void* func(void*)` 的格式

### 3.3 示例代码
```c
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <pthread.h>

void* callback(void* arg)
{
    for (int i = 0; i < 5; i++)
    {
        printf("子线程: i = %d\n", i);
    }
    printf("子线程 ID: %ld\n", pthread_self());
    return NULL;
}

int main()
{
    pthread_t tid;
    pthread_create(&tid, NULL, callback, NULL);

    for (int i = 0; i < 5; i++)
    {
        printf("主线程: i = %d\n", i);
    }
    printf("主线程 ID: %ld\n", pthread_self());

    pthread_exit(NULL);
    return 0;
}
```

## 4. 线程退出

### 4.1 线程退出函数
```c
void pthread_exit(void *retval);
```

作用：
- 终止当前线程
- 不影响同一进程中的其他线程

参数说明：
- `retval`：线程退出时携带的数据，可被 `pthread_join()` 获取
- 如果不需要传值，可写 `NULL`

## 5. 线程回收

### 5.1 线程回收函数
```c
int pthread_join(pthread_t thread, void **retval);
```

作用：
- 阻塞等待指定线程结束
- 回收线程资源

参数说明：
- `thread`：目标线程 ID
- `retval`：接收线程退出时返回的数据；如果不关心，可设为 `NULL`

注意：
- 一个线程只能被回收一次
- 多个线程需要循环 `join`

### 5.2 示例 1：使用全局变量传值
```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

struct Test
{
    int num;
    int age;
};

struct Test t;

void* callback(void* arg)
{
    t.num = 100;
    t.age = 6;
    pthread_exit(&t);
}

int main()
{
    pthread_t tid;
    pthread_create(&tid, NULL, callback, NULL);

    void *ptr = NULL;
    pthread_join(tid, &ptr);

    struct Test* pt = (struct Test*)ptr;
    printf("num = %d, age = %d\n", pt->num, pt->age);
    return 0;
}
```

### 5.3 示例 2：使用主线程栈空间传值
```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

struct Test
{
    int num;
    int age;
};

void* callback(void* arg)
{
    struct Test* t = (struct Test*)arg;
    t->num = 100;
    t->age = 6;
    pthread_exit(t);
}

int main()
{
    struct Test t;
    pthread_t tid;

    pthread_create(&tid, NULL, callback, &t);

    void *ptr = NULL;
    pthread_join(tid, &ptr);

    printf("num = %d, age = %d\n", t.num, t.age);
    return 0;
}
```

## 8. 线程 ID 比较

线程 ID 类型是 `pthread_t`，其底层实现不一定是整数，因此不推荐直接用 `==` 比较。

### 8.1 比较函数
```c
int pthread_equal(pthread_t t1, pthread_t t2);
```

返回值：
- 相等：返回非 0
- 不相等：返回 0

### 8.2 示例
```c
pthread_t tid1 = pthread_self();
pthread_t tid2 = pthread_self();

if (pthread_equal(tid1, tid2))
{
    printf("两个线程ID相等\n");
}
else
{
    printf("两个线程ID不相等\n");
}
```

## 14. 编译线程程序

Linux 下编译线程程序时，需要链接线程库。

推荐写法：

```bash
gcc test.c -o test -pthread
```

也可以写：

```bash
gcc test.c -o test -lpthread
```

更推荐使用 `-pthread`，因为它不仅链接线程库，还会设置线程相关编译选项。
