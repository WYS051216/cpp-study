# 九、生产者消费者模型总结

## 9.1 生产者消费者模型的组成

生产者消费者模型通常包含：

- 生产者线程
- 消费者线程
- 共享缓冲区
- 同步机制

共享缓冲区可以是：

- 链表
- 队列
- 数组
- 环形缓冲区

同步机制可以是：

- 互斥锁 + 条件变量
- 信号量 + 互斥锁

---

## 9.2 条件变量方案

条件变量方案更强调：

> 条件是否满足。

例如：

- 队列是否为空
- 是否有任务
- 是否可以继续执行

典型逻辑：

消费者：

```c
pthread_mutex_lock(&mutex);

while (队列为空)
{
    pthread_cond_wait(&cond, &mutex);
}

/* 消费数据 */

pthread_mutex_unlock(&mutex);
```

生产者：

```c
pthread_mutex_lock(&mutex);

/* 生产数据 */

pthread_mutex_unlock(&mutex);

pthread_cond_signal(&cond);
```

适合：

- 等待某个事件发生
- 等待某个状态改变
- 等待队列不为空

---

## 9.3 信号量方案

信号量方案更强调：

> 资源数量。

例如：

- 缓冲区有几个空位
- 当前有几个产品
- 连接池还有几个连接
- 停车场还有几个车位

典型逻辑：

生产者：

```c
sem_wait(&empty);

pthread_mutex_lock(&mutex);

/* 生产数据 */

pthread_mutex_unlock(&mutex);

sem_post(&full);
```

消费者：

```c
sem_wait(&full);

pthread_mutex_lock(&mutex);

/* 消费数据 */

pthread_mutex_unlock(&mutex);

sem_post(&empty);
```

适合：

- 控制有限资源数量
- 固定容量缓冲区
- 连接池
- 停车位模型
